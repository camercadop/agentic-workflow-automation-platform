# Agent Definitions

Role definitions describing each agent's responsibilities, inputs, outputs, and constraints.

| Agent | Role |
|-------|------|
| `orchestrator.md` | Coordinates the pipeline, invokes agents in sequence |
| `architect.md` | Validates requirements against ADRs, produces design guidance |
| `planner.md` | Breaks requirements into task files with acceptance criteria |
| `developer.md` | Implements code using tool-calling (read/write/run) |
| `tester.md` | Writes and runs tests, measures coverage |
| `reviewer.md` | Reviews implementation against requirements and standards |
